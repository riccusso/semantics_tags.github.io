¿Qué es el HTML semántico y por qué es importante?

Un elemento semántico le indica un significado a lo que se está colocando en el documento

Etiquetas semánticas agregadas en HTML5:

  • < article >
  • < aside >
  • < details >
  • < figcaption >
  • < figure >
  • < footer >
  • < header >
  • < main >
  • < mark >
  • < nav >
  • < section >
  • < sumary >
  • < time >

Cómo funciona el HTML semántico

Indica el tipo de contenido para que el navegador identifique de mejor manera los temas que trata el documento.

Y a su vez es mejor para leer del punto de vista del usuario para leer de que tarata cada parrafo o tema.
